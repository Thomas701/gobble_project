int createCase ( SDL_Rect ***  pTableauCase)

Permet de créer dynamiquement le tableau de case pour l'interface graphique.
.

Chaque case du tableau est un pointeur vers un case de la map.

Paramètres
[in]SDL_Rect*** pTableauCase : Prend un pointeur vers un tableau de case car va modifier les coordonées de ce dernier.
Renvoie
int : Retourne entier pour savoir si tout c'est bien passé. -1 si erreur, 0 sinon
