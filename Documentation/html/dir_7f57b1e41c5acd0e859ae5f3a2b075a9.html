Fichiers

fichier  creation.h [code]
 Fichier contenant les headers des fonctions permettant la créations des piles, plateau, tableau de cases et de coordonées des cases.
 
fichier  demande.h [code]
 Fichier contenant les headers des fonctions où l'on demande à l'utilisateurs ces choix d'action au clavier. Avec vérification.
.
 
fichier  error.h [code]
 Fichier contenant les headers des fonctions d'erreurs (affiches messages et quitte le programme).
 
fichier  fonction.h [code]
 Fichier contenant les headers des fonctions "actives" liées aux mouvement des pions.
 
fichier  free.h [code]
 Fichier contenant les headers des fonctions permettant de free de nos structures de données commes les piles, plateaux, tableaux, textures.
 
fichier  get.h [code]
 Fichier contenant les codes des fonctions permettant d'écrire et lire les maps et piles dans des fichiers. On ne les utilise pas.
 
fichier  graphique.h [code]
 Fichier contenant les headers des fonctions graphiques. Elles gèrent le chargement des images (initialisations) et leurs affichagent.
 
fichier  init.h [code]
 Fichier contenant les headers des fonctions permettant l'initialisation de nos structures de données commes les piles, plateaux, tableaux, textures.
 
fichier  print.h [code]
 Fichier contenant les headers des fonctions permettant d'afficher dans le terminal le plateau et les piles.
 
fichier  saveFile.h [code]
 Fichier contenant les headers des fonctions permettant d'écrire et lire les maps et piles dans des fichiers. On ne les utilise pas.
 
fichier  verification.h [code]
 Fichier contenant les headers des fonctions permettant de vérifier si un déplacement / insertion de nouveau pion est valide.
