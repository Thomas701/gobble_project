◆ moove()

int moove ( char ***  map,
int  posDeb,
int  posEnd 
)

Permet de déplacer un pion se trouvant sur la map.
.

Il faut vérifié avant si le déplacement est valide, mais sur-vérification grâce à canMoove().

Paramètres
[in]char*** map : plateau de jeu détaillé
[in]intposDeb : position de départ du pion (entre 0 et 8 si 3*3)
[in]intposEnd : position d'arriver du pion (entre 0 et 8 si 3*3)
Renvoie
1 si on a réussi a mettre le pion sur la map \retrun 0 si déplacement non valide (canMoove) ou si erreur lors de l'insertion (n'arrive jamais).

Permet de placer le pion sur la map. Il faut d'abord vérifier si le coup est valide.

Paramètres
[in]char*** map : plateau de jeu détaillé
[in]char** stackArray : pile du joueur qui déplace
[in]intnumStack : numéro de la pile du joueur (exemple si 3*3 seulement 2 piles donc numéro 0 ou 1)
[in]intsizePion : taille du pion à déplacer(permet de l'identifier au sein de la pile)
[in]intendPion : position sur la map ou il faut mettre le pion (entre 0 et 8 si 3*3)
[in]charc : lettre qu'il faut mettre sur la map, soit 'b' si joueur 1, sinon 'n'
Renvoie
Pas de retour (void).
Auteur
DUPOIS Thomas & VILLEPREUX Thibault
