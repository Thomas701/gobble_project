Fonction qui gère lorsque le joueur se trouve dans le menu principale.
.

Il peut lancer une partie 1VS1, 1VSIA, quitter le jeu (autre que par la croix rouge) ou couper et rejouer le son.

Paramètres
[in]SDL_Renderer** renderer : rendu ou l'on colle les images.
[in]SDL_Texture** textureBackground : Tableau de texture d'images qui va contenir les 400 images d'animations.
[in]SDL_Texture** textureTableauOptionMenu : Tableau de texture des images des options menu (1VS1, 1VSIA, SOUN , QUIT GAME)
[in]int* p_etats : pointeur permettant de connaître l'état du programme (en game, en pause)
[in]intboolPlayMusic : permet de savoir si on joue de la musique ou pas.
[in]int* ia : pointeur sur l'ia (permet de définir s'il y'aura une ia ou pas)
Renvoie
void : Pas de valeur de retour.

DUPOIS Thomas

SDL_Texture* loadImage ( const char *  path,

)

Fonction qui va permmetre de transformer une image (n'importe quel format valide) en texture.

Paramètres
[in]constchar * path : prend le chemin de l'image à charger.
[in]SDL_Renderer*renderer : associe la texture créer à un renderer.
Renvoie
SDL_Texture * : Retourne un pointeur vers la texture à manipuler.
NULL : Si chemin d'accès invalide ou erreur allocation texture à cause de SDL_CreateTextureFromSurface (message d'erreur qui prévient)
