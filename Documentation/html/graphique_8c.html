Fonction qui lance la partie Homme VS Homme.

Elle appelle gameOptionGraphique qui joue la partie et ensuite elle affiche le vainquer et réinitialise plateau et pile pour prochaine partie.

Paramètres
[in]SDL_Renderer** renderer : rendu ou l'on colle les images.
[in]SDL_Texture** tableauTextureMapVide : tableau de texture d'images qui contient map vide bleau ou rouge selon le tour du joueur.
[in]point** tableauDePoint : tableau de point pour gérer l'affichage des pions.
[in]SDL_Texture** textureTableauWin : tableau de texture, contient les images de victoires avec lignes gagnantes éclairées.
[in]int* p_etatS : pointeur permettant de connaître l'état du programme (en game, en pause)
[in]SDL_Rect** tableauCase : tableau de case / surface pour gérer l'affichage des pions et leurs sélections.
[in]SDL_Texture** textureTableauPion : tableau de texture, contient les images des pions sélectionnés et non selectionnés.
[in]char*** map3D : contient la map du jeu détaillé.
[in]char** map2D : contient la map du jeu non détaillé.
[in]char** pileJ1 : contient la pile du joueur 1.
[in]char** pileJ2 : contient la pile du joueur 2.
[in]intdeplacement : option de jeu avec deplacement restreins ou non .
Renvoie
void : Pas de valeur de retour.

Fonction qui gère lorsque le joueur se trouve dans le menu principale.
.

Il peut lancer une partie 1VS1, 1VSIA, quitter le jeu (autre que par la croix rouge) ou couper et rejouer le son.

Paramètres
[in]SDL_Renderer** renderer : rendu ou l'on colle les images.
[in]SDL_Texture** textureBackground : Tableau de texture d'images qui va contenir les 400 images d'animations.
[in]SDL_Texture** textureTableauOptionMenu : Tableau de texture des images des options menu (1VS1, 1VSIA, SOUN , QUIT GAME)
[in]int* p_etats : pointeur permettant de connaître l'état du programme (en game, en pause)
[in]intboolPlayMusic : permet de savoir si on joue de la musique ou pas.
Renvoie
void : Pas de valeur de retour.
