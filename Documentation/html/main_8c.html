Fonction principale, entrée du programme.

Paramètres
[in]intargc : compte le nombre de paramètre de l'éxecutable.
[in]char** argv : stocke sous forme de chaine de caractères les paramètres données lors de l'éxecution.
Par exemple "./main -c" ou "./main --console" lance le jeu uniquement dans le terminal.
Par défault, il se lance en interface graphique.
Renvoie
EXIT_SUCCESS - Arrêt normal du programme.
EXIT_FAILURE - Arrêt anormal du programme (probleme provenant dans la plus cas des cas de l'initialisation de la SDL2 ou des textures).
